# Prompt — Descrição do Produto

Antes de responder, leia toda esta conversa do início ao fim.

Utilize todas as informações já existentes como contexto.

Nunca peça novamente informações que já foram fornecidas.

Faça perguntas apenas quando identificar uma lacuna indispensável para executar a tarefa.

---

Utilizando todo o contexto desta conversa, crie uma descrição completa e persuasiva para o meu produto digital.

Primeiro analise:

- quem eu sou;
- quem é meu público;
- produto criado;
- promessa;
- transformação;
- benefícios;
- módulos;
- diferenciais;
- formato do produto;
- oferta.

Depois gere:

1. Descrição curta (até 250 caracteres).
2. Descrição completa.
3. Principais benefícios.
4. O que o aluno receberá.
5. Para quem este produto é indicado.
6. Diferenciais do produto.

Escreva de forma clara, natural e persuasiva.

Não invente informações.

Caso alguma informação realmente esteja faltando, faça apenas uma pergunta antes de gerar.
