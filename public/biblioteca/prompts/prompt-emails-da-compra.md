# Prompt — E-mails da Compra

Antes de responder, leia toda esta conversa do início ao fim.

Utilize todas as informações já existentes como contexto.

Nunca peça novamente informações que já foram fornecidas.

Faça perguntas apenas quando identificar uma lacuna indispensável para executar a tarefa.

---

Utilizando todo o contexto desta conversa, gere todos os e-mails necessários para este produto.

Analise primeiro:

- produto;
- público;
- promessa;
- transformação;
- linguagem;
- personalidade da marca.

Depois gere:

1. E-mail de confirmação da compra.
2. E-mail de boas-vindas.
3. E-mail orientando os primeiros passos.
4. E-mail de recuperação de carrinho.

Todos os e-mails devem ser personalizados para este produto.

Utilize o mesmo tom de comunicação adotado durante toda a criação do produto.

Escreva textos naturais, humanos e acolhedores.

Inclua assuntos com alta taxa de abertura e chamadas para ação coerentes.

Não invente informações.

Caso alguma informação realmente esteja faltando, faça apenas uma pergunta antes de gerar.
